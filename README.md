# CSV Performance Report

Скрипт, который читает один или несколько CSV файлов с данными о задачах сотрудников и считает среднюю эффективность по каждой позиции.

## Как запустить

```bash
python main.py --files <файл1.csv> [файл2.csv ...] --report performance
```

Аргумент `--report` оставлен для совместимости, скрипт поддерживает только отчет `performance`.

## Формат CSV

Файлы должны содержать хотя бы колонки:

- `position` — позиция сотрудника
- `performance` — числовое значение эффективности


## Пример вывода

```
+--------------------+-------------------------+
| Позиция            |   Средняя эффективность |
+====================+=========================+
| DevOps Engineer    |                    4.95 |
| Backend Developer  |                    4.85 |
| Frontend Developer |                    4.75 |
| Data Scientist     |                    4.65 |
| QA Engineer        |                    4.55 |
+--------------------+-------------------------+
```
